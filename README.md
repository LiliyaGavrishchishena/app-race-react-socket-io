## app-race-react-socket-io

### Setting up Dev

Clone the project:

```sh
git clone https://github.com/LiliyaGavrishchishena/app-race-react-socket-io.git
```

```sh
cd app-race-react-socket-io
```

Install all dependencies and run:

```sh
npm install
```

```sh
npm start
```

### Building

```sh
npm run build
```

### Deploying

```sh
npm run deploy
```
